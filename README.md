# Google-Expense-Sharing-
The Expense Sharing Application is a Python-based project that helps users manage and split shared expenses among friends, family members, roommates, or colleagues. Similar to the expense-sharing feature in Google Pay, the application allows users to record expenses, divide costs equally or custom-wise, and track who owes money to whom.
Features
Add multiple users/friends dynamically.
Record shared expenses.
Equal expense splitting among participants.
Automatic balance calculation.
Settlement tracking (who pays whom).
Transaction history management.
Tabular output using PrettyTable.
Efficient calculations using NumPy.
Technologies Used
Python – Main programming language.
NumPy – For matrix-based expense calculations.
PrettyTable – For displaying expense and settlement data in a structured table format.
SQLite (Optional) – For storing transaction records permanently.
Use Case


Instead of manually calculating each person's share, the application:

Splits expenses among selected participants.
Tracks how much each person paid.
Calculates how much each person owes.
Generates a final settlement report showing who should pay whom
